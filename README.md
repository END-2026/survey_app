# 📝 Опросник: Отношение к анонимным отзывам в сети

**Вариант №46**  
Учебный проект по созданию одностраничного опросника на Python + Streamlit + Firebase.

---

## 🎯 Тема исследования

Исследование достоверности, влияния и модерации анонимных отзывов в интернете.

**Ключевые вопросы:**
- Насколько пользователи доверяют анонимным отзывам?
- Влияют ли они на принятие решений?
- Сталкивались ли респонденты с фейковыми отзывами?
- Как они оценивают необходимость модерации?

---

## ✨ Функциональность

### Для пользователей:
- 📋 Анонимная форма из 8 вопросов
- 💾 Автоматическое сохранение ответов в облачную БД
- 🔒 Безопасное хранение данных

### Для преподавателя/исследователя:
- 📊 5 интерактивных графиков (Plotly)
- 📋 Таблица со всеми собранными ответами
- 🔄 Автоматическое обновление аналитики

### Графики:
1. Гистограмма уровня доверия (1–10)
2. Влияние отзывов на решение
3. Опыт встречи с фейками (круговая диаграмма)
4. Отношение к модерации
5. Популярные платформы для чтения отзывов
